MATLAB Routines¶

msrc.mwright(x, t, lambda, mu, varargin)¶

MWRIGHT This routine computes the Wright function on the real line: \(t^{-\lambda} W_{\lambda,\mu}(-|x|t^{-\lambda})\)

param

x (may be a vector)

param

t has to be a scalar > 0

param

-1 < lambda < 0

param

mu scalar

param

Optional, the number of quadrature points N

Returns

w and number of used quadrature points N
